# Portfolio Website

This repository contains a personal portfolio website for Poonam Bora.

## What was done

- Split the original single-file HTML into separate files:
  - `index.html` for the page structure and content
  - `styles.css` for all styling rules
  - `script.js` for the navigation menu and reveal animations

- Updated `index.html` to link the new CSS and JavaScript files.

## Notes

- All files are located in the same folder for a simple project structure.
- The page was kept functionally identical while separating concerns.
